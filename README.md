# SMS Bulk TZ

## Overview
This repository provides tools for sending bulk SMS messages to multiple recipients efficiently.

## Features
- User-friendly interface
- Support for multiple SMS gateways
- Bulk sending capabilities

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/Lazack28/sms-bulk-tz.git
   ```
2. Install dependencies:
   ```bash
   cd sms-bulk-tz
   npm install
   ```
3. Configure your SMS gateway settings in the `config.js` file.

## Usage
To send bulk SMS:
```bash
node send.js --message "Your message here" --numbers "num1,num2,num3"
```
